\# OK Decision-Making Style Simulator



An interactive psychometric-style web simulator that analyzes user decision-making patterns based on scenario-based inputs.



\## Features

\- Scenario-driven behavioral simulation

\- Decision-style classification algorithm

\- Real-time result computation

\- Clean responsive UI

\- No backend required



\## Technologies Used

\- HTML5

\- CSS3

\- JavaScript (Vanilla JS)



\## Concept

This simulator models cognitive decision frameworks and classifies users into dominant decision-making styles such as Analytical, Intuitive, Impulsive, or Avoidant.



Developed for Science Expo 2026.
